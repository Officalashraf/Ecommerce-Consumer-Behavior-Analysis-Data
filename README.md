# 🛒 Ecommerce Consumer Behavior Analysis

## 🌟 Projektübersicht (Overview)
In diesem Projekt werden umfangreiche E-Commerce-Transaktionsdaten analysiert, um das Kaufverhalten der Kunden zu verstehen. Ziel ist es, Muster wie Kaufhäufigkeit, Warenkorb-Abbruchraten und Produktpräferenzen zu identifizieren, um datengesteuerte Entscheidungen für Marketing und Bestandsmanagement zu treffen.

## 🚀 Kernfunktionen (Key Features)
*   **RFM-Analyse (Recency, Frequency, Monetary):** Segmentierung der Kunden in Gruppen (z. B. "Champions", "Abwanderungsgefährdet"), um gezielte Marketingstrategien zu entwickeln.
*   **Warenkorbanalyse (Market Basket Analysis):** Identifizierung von Produkten, die häufig zusammen gekauft werden (Association Rules).
*   **Zeitreihenanalyse:** Untersuchung von Verkaufstrends nach Tageszeit, Wochentag und Saison.
*   **Customer Lifetime Value (CLV):** Schätzung des zukünftigen Wertes von Kunden für das Unternehmen.

## 📊 Analyse-Schwerpunkte (Analysis Highlights)
*   Welche Produkte haben die höchste Wiederkaufsrate?
*   Zu welchen Uhrzeiten ist die Conversion-Rate am höchsten?
*   Identifizierung von Clustern innerhalb der Kundenbasis basierend auf dem Ausgabeverhalten.

## 🛠️ Tech Stack
*   **Sprache:** Python
*   **Datenanalyse:** Pandas, NumPy
*   **Visualisierung:** Plotly (Interaktive Charts), Seaborn, Matplotlib
*   **Machine Learning:** Scikit-learn (K-Means Clustering für Kundensegmentierung)
